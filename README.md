# CareerForge AI 🚀

AI-Powered Career Copilot built using React.js, Node.js, Express.js, MongoDB Atlas, and OpenAI API.

## Features

- Resume Analyzer
- ATS Score Checker
- Job Match Analyzer
- Skill Gap Analysis
- AI Career Roadmap Generator
- Interview Preparation
- Job Application Tracker
- Analytics Dashboard
- Profile Management

## Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### AI Integration
- OpenAI API

### Tools
- Git
- GitHub
- Postman

## Installation

```bash
git clone https://github.com/tejachaganti/CareerForge-AI.git

cd CareerForge-AI

npm install

npm run dev
```
